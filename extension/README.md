# Multiplay Tier Maker Apply Tool

TierMaker `create/...` 페이지 오른쪽에 플로팅 메뉴를 띄워 현재 템플릿을 Multiplay Tier Maker 서버로 보냅니다.

## 설치

1. Chrome 또는 Edge에서 `chrome://extensions`를 엽니다.
2. `개발자 모드`를 켭니다.
3. `압축해제된 확장 프로그램을 로드`를 누릅니다.
4. 이 폴더를 선택합니다: `E:\MULTITIERMAKER\extension`

## 사용

1. `dist\MultiplayTierMaker.exe` 또는 `npm start`로 서버를 켭니다.
2. TierMaker `create/...` 페이지를 엽니다.
3. 오른쪽 `Multiplay Tier Maker` 패널에서 서버 주소를 확인합니다.
4. `현재 템플릿 적용`을 누릅니다.

서버 주소 기본값은 `http://localhost:3000`입니다. Cloudflare Tunnel이나 Render 주소를 쓰는 경우 패널의 서버 주소를 해당 URL로 바꿉니다.
